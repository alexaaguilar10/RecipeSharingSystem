# RecipeSharingSystem
## How to Test Different Architectures:
1. Clone the repository if the user hasn’t already.
- git clone <repository_url>
- cd <repository_folder>

2. Switch to the branch:
- git checkout mvc-architecture
OR
- git checkout layered-architecture
OR
- git checkout event-driven_architecture

3. Install dependencies (if applicable):
- npm install   # or pip install -r requirements.txt, etc.

4. Run the project:
- npm start
